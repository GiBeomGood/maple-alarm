1. `cargo release --build` 해서 나온 `target/release/alarm-app.exe` 파일 실행하면, 시스템 트레이에 아이콘이 안 나오고 빨간 네모로만 나옴
2. 시스템 트레이에서 우클릭하고 종료하면 바로 안 꺼지고 딜레이 좀 있고 꺼짐. 왜 늦게 꺼지지? 생각보다 많이 느림.
3. 알람이 울리지 않을 때에도, 확인 버튼을 누르면 타이머가 리셋되면 좋을 듯. 아직 시간 남아 있는데 확인 누르면 1분 40초 (혹은 설정된 시간) 리셋되게.
4. 알람 소리 울리는 타이밍이랑 색깔 초록색으로 바뀌는 타이밍이랑 좀 다른 듯. 소리 울림과 동시에 초록색이 빨간색으로 바뀌고, 확인 버튼이 활성화되면 좋겠음.
